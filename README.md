# Multi-Agent Research Scientist

An AI system that autonomously conducts academic research using a pipeline of 5 specialized agents.

**Built with:** LangGraph · Anthropic Claude · ChromaDB · FastAPI · React

## Agents
1. **Planner** — Breaks down the research question into search queries
2. **Search** — Retrieves papers from arXiv and Semantic Scholar
3. **Critic** — Evaluates evidence quality and finds contradictions
4. **Writer** — Synthesizes a structured research report
5. **Fact Checker** — Verifies claims against source evidence
